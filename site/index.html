Hold the majors in uptrends. Cash in downtrends.

A CoinMarketCap skill that holds the major coins that are trending — BTC, ETH, BNB, SOL — and steps aside to stablecoins for the rest. We tested 27 strategies; this one caught the rallies and dodged the −50% crashes.

Beat 02 — What it is

Each major above its trend: hold it. Below: cash.

Every major coin gets one line — its own moving average. Above it, the agent holds that coin; below it, that slice sits in stablecoins. Equal-weight whatever's trending. That's the entire rule.

A coin above its average
Hold it
A coin below its average
That slice → cash
Result, 2023–26 (100-day)
+1945% vs +254% just holding the four — at a third of the drawdown

It never nails the exact top or bottom — it just stays in for the big uptrends and out for the deep crashes, spread across four liquid majors so no single coin can sink it. Over years, that's what compounds.

The research

We tested 27 strategies. Trend-following won.

We backtested 27 strategies across 7 families on the same data, then split the timeline: tune on the first half (in-sample), score on the unseen second half (out-of-sample). A strategy only counts if it works on data it never saw.

Get it running

Install in any Claude-skill host.

Drop the skill folder into your skills directory:

cp -r skills/trend-regime /path/to/your/skills/directory/

Then connect the CoinMarketCap MCP (get a free key at pro.coinmarketcap.com):

{
  "mcpServers": {
    "cmc-mcp": {
      "url": "https://mcp.coinmarketcap.com/mcp",
      "headers": { "X-CMC-MCP-API-KEY": "your-api-key" }
    }
  }
}

Then just ask: Which majors are trending right now?  or  /trend-regime

Hands-off

Or hand it to an agent on Vercel.

Run it by hand, or let one scheduled function do the whole loop — nobody triggers a trade manually.

01

Vercel Cron fires daily and calls one serverless function. That's the only trigger.

02

It pulls each major's price + moving average from the CoinMarketCap MCP (via the Vercel AI SDK), runs the rule in code — not the model, so it's reproducible — and sizes the equal-weight book.

03

On a regime flip only, it sends a Telegram alert and — if you opt in — executes the swap. It never holds your keys.

// vercel.json — the only scheduler
"crons": [{ "path": "/api/trend-tick", "schedule": "0 12 * * *" }]

// api/trend-tick.js — the decision is plain code, not the LLM
for (const c of MAJORS)
  target[c] = price[c] > movingAvg[c] ? "HOLD" : "CASH";

Built on Vercel's agent stack — Cron, Functions, the AI SDK MCP client and the AI Gateway. The demo runs notify-only (paper trade); live execution is the same code path, opt-in.

The engine

Built to run on Claude Opus 4.8.

Why this model for an always-on agent that touches money? Three things matter more than raw IQ:

01

Reliable tool use. Opus 4.8 specifically improved tool triggering — it doesn't skip the price, balance, or execute calls a trading tick depends on.

02

Long-horizon stability. A 1M-token context and steadier long agent loops mean it doesn't derail over a run that lives for months.

03

Honest about failure. It glosses over hidden failures only ~3.7% of the time (per Anthropic's Opus 4.8 announcement) — a money agent has to surface bad data and failed fills, not bury them.

And it's deterministic by design: structured JSON outputs, behavior driven by explicit rules rather than random sampling. The model orchestrates and explains; the strategy stays a hard rule. More on Claude Opus 4.8 →

Not financial advice — you bear your own risk

This project is an educational research tool built for a hackathon. Nothing here is financial, investment, or trading advice, and nothing is a recommendation to buy, sell, or hold any asset. You are solely responsible for your own decisions, your own funds, and your own risk.

Trend-following is not risk-free. It whipsaws in choppy markets — shaken out low, bought back high, bleeding small losses. It always lags the exact top and bottom. The backtest uses daily Binance data with costs applied; live slippage, gas, and timing differ. And 2023–2026 was largely a bull market, which flatters any long/trend strategy — the headline returns will not repeat.

Out-of-sample testing reduces overfitting risk but doesn't remove it: we tested 27 strategies, and the more you test, the easier a lucky one slips through. We guard against it by requiring a whole family to hold up on unseen data — but no backtest guarantees the future. Past performance does not predict future results.

Do your own research. Only deploy capital you can afford to lose.
